Elementos HTML

<p>Isto é um parágrafo </p>

Um elemento HTML é marcado por tags, como o <p> acima; Conteúdo do elemento, "Isto é um parágrafo"; e geralmente uma tag de fechamento </p>

estrutura do HTML

As páginas HTML devem possuir uma estrutura base:

<!-- Comentários no HTML são feitos dessa forma -->

<!DOCTYPE html> <!-- informa sobre o tipo de html que será utilizado -->

<html> <!-- representa a raiz do documento (todo código HTML deve estar entre ela) -->

<head> </head> <!-- Indica metadados sobre a página, como título, ícone, etc -->

<body> </body> <!-- Parte visual da página, parágrafos, tabelas, imagens -->

</html>

Básicos do HTML

Algumas tags básicas utilizadas no HTML

Título da página

Você pode definir o nome da página dentro da tag <title> que fica dentro da tag <head>

Títulos e subitítulos

Existem tags de <h1> até <h6> e elas são utilizadas para títulos e subtítulos

Parágrafos

A tag <p> indica que o texto faz parte de um parágrafo

Atributos no HTML

Os elementos HTML podem ter atributos que dão informações adicionais para o elemento. São sempre colocados nas tags de abertura e geralmente são assim: atributo="valor" . O HTML possui dois tipos de atributos:

  1. Globais: É possivel de utliza-los em todos elementos HTML (id e class)
  2. Específicos: Só é possivel utiliza-lo de acordo com o elemento usado

Formatação de texto

É possivel deixar alguns tipos especiais de texto

<b> e <strong>

Ambos conseguem deixar o texto em negrito, porém a tag strong indica maior importância

<i> e <em>

Ambos deixa o itálico, as diferenças são que a tag i é utilizada para mostrar termos técnicos, pensamentos ou outra linguagem enquanto que a tag em dá uma ênfase na palavra, quando um leitor de tela ler um elemento com a tag em ele será lido com mais ênfase

<mark>

A tag mark indica que um texto deve ser marcado ou destacado

<small>

Simplesmente indica um texto com uma fonte menor

<del>

Indica que um texto foi adicionado deletado

<sub> e <sup>

Ambas são parecidas, enquanto a sub é um texto colocado na parte inferior do texto a sup é colocado na parte superior

Geralmente são utilizadas para escrever fórmulas, como: H2O e E=MC2

Listas em HTML

Há três tipos de listas em HTML, listas ordenadas (<ol>), não ordenadas (<ul>) e listas de descrição, para inserir a lista basta colocar entre as tags do tipo de lista que deseja e colocar a tag <li>(list item) para cada item da lista.

Lista ordenada <ol>:

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

É possivel começar a lista de qualquer número com o atributo start=""

Lista não ordenada <ul>:

  • Elemento
  • Elemento
  • Elemento

A propriedade list-style-type no CSS é usada para definir o estilo da marcação de item. possiveis valores:

  • disc - Define o marcador como um disco (padrão)
  • circle - Define o marcador como um círculo
  • square - Define o marcador como um quadrado
  • none - Remove o marcador

Lista descritiva <dl>:

Coffee
- black hot drink
Milk (<dt>)
- white cold drink (<dd>)

Inserção de imagem

Para inserir uma imagem em HTML basta abrir a tag <img> e colocar no atributo src="" o diretório onde está localizado a imagem, como feito no print acima.

O atributo src permite que você coloque imagem na pasta do projeto e em diferentes servidores/sites colocando a url.

Também é possivel colocar gifs no atributo src.

Atributos adcionais

  • alt="" - Texto alternativo caso a imagem não seja carregada.
  • style="" - Pode ser usado para definir altura e largura da imagem.
    • width:500px; - Define uma largura de 500 pixels.
    • height:500px; - Define uma altura de 500 pixels.
    • width e height tambem podem ser usados como atributos separados.

<picture>

A tag picture permite você colocar diferentes imagens para diferentes dispositivos

O elemento picture possui um ou mais elementos <source>, cada um direcionando para um imagem através do atributo srcset="". Cada elemento <source> também possui um atributo media que define quando a imagem deve ser inserida.

Sempre use uma <img> no fim da <picture>. Ela será usada caso nenhuma imagem do <source> for suportada.

Favicon

Favicon é a imagem mostrada no topo do navagdor ao lado do título da página. Um favicon pode ser criado em Favicon.cc.

Pode ser inserido usando a tag <link> da seguinte maneira:

<link rel="icon" type="image/x-icon" href="/images/favicon.ico">

Links em HTML

Para inserir links, é necessário utilizar a tag <a> com atributos de href="" apontando para onde deverá ser redirecionado, ex: Site do google (href="https://google.com")

É possivel Utilizar como valores no atributo href="":

  • URL absolutas: aponta para outro site (https://www.w3schools.com)
  • URL relativa: aponta para um site dentro de alguma pasta (pagina2.html)
  • Para um elemento com id específico na própria página (#idElemento)
  • Outros protocolos (https://, ftp://, mailto:, file:, etc..)
  • Um script (href="javascript:alert('Hello');")

Atributos adcionais

  • target="" - Indica como a página referenciada no link será aberta
    • _blank - Abre a página indicada em nova guia
    • _self - Abre na mesma página
  • title="" - Informação extra ao passar mouse sobre link

Tabelas

Nome Idade
Joao Pedro 1.66
Heitor
<table>
 <colgroup>

 </colgroup>
  <tr>

    <td>20</td>

   <td></td>

Cada conteúdo da célula da tabela fica entre as tags td (table data)

Cada linha da tabela é definida pela tag tr (table row), É possivel ter quantas linhas quiser, porém tomar cuidado para ter mesmo número de células em cada

Você pode utilizar th (table header) para células de cabeçalho, que por padrão são centrilizadas e estão em negrito

rowspan e colspan

É possivel ocupar 2 ou mais espaços para linhas e colunas na tabela

Com <colspan> você coonsegue definir quantos espaços na mesma linha o atrbito irá ocupar

<rowspan> segue a mesma ideia, porem para colunas.

colgroup

É possivel agrupar um conjunto de colunas com a tag <colgroup> e <col> com atributo span indicando quantas colunas serão agrupadas, esses elementos precisam vir antes das tags <th>, <td> e <td>

Pórem há prorpriedades limitadas do CSS para se utilizar em um <colgroup>:

  • width (largura)
  • visibility (visibility)
  • background (imagem e cores de fundo)
  • border (bordas)

Se deseja agrupar colunas separadas basta criar tags col com span vazios entre os grupos

Block e inline

Todo elemento HTML possui um valor display padrão; block e inline são os mais comuns

Block

Elementos block sempre começam em uma nova linha e ocupam a máxima largura que conseguirem por padrão

Exemplos de elementos block:

<adress> <table> <footer> <h1>-<h6>
<section> <aside> <video> <p>
<article> <main> <form> <header>
<li> <nav> <div> <pre>

Inline

Elementos inline não iniciam uma nova linha e ocupam somente a largura necessária

Exemplos de elementos inline:

<a> <span> <strong> <b>
<sup> <i> <img> <input>

<div>

Geralmente é um container para outros elementos HTML, não possui atributos além de style, class e id. Pode ser usado como um bloc de conteúdo

Se quiser alinhar uma div que não ocupe toda sua largura, basta usar propriedade do CSS margin:auto; - Margin no auto fará com que o elemento possua a maior margin horizontal possivel, consequentemente, centralizando o elemento

<span>

Elemento inline que é usado para marcar um texto ou parte de um conteúdo, também não possui elementos além de style, class e id. Pode ser usado para estilizar partes de um texto

Inline-block

Se voce colocar a propriedade display com valor "inline-block" os elementos div não vao mais quebrar uma linha antes e dpois e será mostrado lado a lado ao inves de verticalmente

flex

display: flex; foi introduzido para facilitar a fazer designs flexíveis responsivos.

Para fazer o flex funcionar, rodeie a div por outra div e dê a div pai o display: flex;

Atributos class e id

São atributos usados para especificar elementos no HTML

Class

Geralmente é usado para um nome de classe no CSS, também pode ser usado no JavaScript para acessar e manipular os elementos.

Os elementos podem possuir mais de uma classe, separados por espaço. Diferentes elementos podem compartilhar as mesmas classes

id

É usado para especificar um único elemento HTML. ou seja, deve ser único por elemento.

Iframe

tag iframe é utilizada para colocar um site dentro de outro site

<iframe srv=css.html height="300px" width="1400px">

Tag head

A tag head contem elementos que não necessariamente vão aparecer na página, possui informações sobre a própria página (metadados). pode conter as tags <title>, <style>, <meta>, <link>, <script>, and <base>.

title

A tag title define o título do documento, e é obrigatório em um documento HTML. o título:

  • Define um título para a página
  • Coloca um título para quando página for adcionada aos favoritos
  • Mostra o tpitulo em páginas de pesquisa

link

A tag link define uma relação entre o documento atual e algum recurso externo

meta

A tag meta é usada para especificar o character set, descrição da página, teclados, autor do documento e configurações de visualização

Configurando viewport

viewport é a área visivel de uma página, que varia de acordo com os dispositivos. Voce deve incluir o seguinte comando em todas suas páginas:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

O width=device-width coloca a largura da página para seguir a largura do dispositivo

A initial-scale=1.0 define o level de zoom inicial da página quando a página for carregada

Layout

Website HTML podem conter contepúdos em varias colunas, como em revistas e jornais

Técnicas para criar layouts com várias colunas (cada uma possuindo seus prós e contras)

  • Frameworks css
  • Propriedade float
  • flexbox
  • grid

Frameworks

Se quiser criar seus layouts de forma rápida, você pode usar o um framework CSS, como o Bootstrap

float

É comum fazer layouts web utilizando o float. só é necessário entender como as propridades float e clear funcionam.

Disvantagens: elementos float estão amarrados com o movimentos do documento, e isso pode afetar a flexiblidade

flexbox

Uso de flexbox garante que elementos se comportem de maneira previsível quando o layout precisar acomodar diferentes tamanhos de tela

grid

O uso do módulo grid oderece um layout em grade, com linhas e colunas, ficando mais fácil para montar páginas web sem usar float

Elementos semântivos

São elementos que possuem um significado, tanto para o navegador tanto para o desenvolvedor, mas não interferem na estrutura do HTML

Alguns exemplos:

  • <header> - Container para introdução
  • <main> - Indica onde está o principal conteúdo da página
  • <nav> - Define uma navegação entre sites e a página
  • <article> - Conteúdo independente, auto-explicativo
